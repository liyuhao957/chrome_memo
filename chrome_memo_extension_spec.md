# 网站备忘录 Chrome 扩展规格说明书

## 1. 项目概述

开发一个 Chrome 浏览器扩展，允许用户为不同网站创建和显示备忘录。当用户访问特定网站时，相应的备忘录将自动显示在页面上。扩展支持模板功能、快捷键操作和选中文本快速添加等功能。

## 2. 核心功能

### 2.1 基础功能
- 支持多个网站的备忘录功能
- 基于域名关联备忘录内容
- 使用本地存储，支持离线工作
- 页面右下角显示可拖拽的悬浮窗形式备忘录
- 每个域名对应一条备忘录
- 访问网站时自动显示对应备忘录

### 2.2 内容与编辑
- 支持富文本格式（加粗、斜体、下划线、链接等）
- 备忘录内容编辑器具有工具栏
- 支持选中网页文本快速添加到备忘录
- 支持引用块样式的文本格式

### 2.3 用户交互
- 通过右键菜单、扩展弹出窗口或快捷键触发功能
- 支持隐藏/显示备忘录的切换功能
- 备忘录支持拖拽定位，可保存位置
- 支持最小化为悬浮图标
- 双击备忘录内容进入编辑模式

### 2.4 快捷键支持
- Alt+M (Mac: Option+M): 显示/隐藏备忘录
- Alt+E (Mac: Option+E): 打开备忘录编辑器
- Alt+Q (Mac: Option+Q): 开启/关闭选中文本添加功能

### 2.5 模板功能
- 将当前备忘录内容保存为模板
- 从编辑器中快速应用已保存的模板
- 支持模板拖拽排序
- 模板在所有网站间共享，实现跨站点使用

## 3. 技术实现

### 3.1 架构组成

#### 3.1.1 背景脚本 (Background Script)
- 处理右键菜单操作
- 管理扩展生命周期
- 处理快捷键命令
- 管理模板数据存储和访问
- 处理跨组件消息通信

#### 3.1.2 内容脚本 (Content Script)
- 在网页中插入备忘录组件
- 处理备忘录的显示、隐藏和编辑
- 实现拖拽定位功能
- 处理选中文本快速添加功能
- 实现富文本编辑器
- 处理模板应用

#### 3.1.3 弹出界面 (Popup)
- 显示当前页面备忘录状态
- 提供快速操作按钮
- 显示已保存的备忘录站点列表
- 支持模板快速选择和应用
- 提供备忘录批量管理功能

### 3.2 数据结构

#### 3.2.1 备忘录数据
```javascript
{
  memos: {
    [domain]: {
      content: string,      // HTML内容
      isVisible: boolean,   // 显示状态
      lastEdited: number,   // 最后编辑时间戳
      position: {           // 位置信息
        left: string,
        top: string,
        right: string,
        bottom: string
      }
    }
  }
}
```

#### 3.2.2 模板数据
```javascript
{
  templates: {
    [templateName]: {
      content: string,      // HTML内容
      createdAt: number,    // 创建时间戳
      order: number,        // 排序位置
      domain: string        // 来源域名
    }
  }
}
```
